# Demo coverage

The walkthrough follows the app end-to-end in line with the **Sample interactions** in [`README.md`](../README.md):

1. **Household** — demo household and tasks; natural-language task entry (with API key when showing AI).
2. **My Schedule** — **Next 14 days**; **Build today’s plan**; time planned vs available; **Today’s checklist**.
3. **Ask PawPal** — scheduling question with **Let PawPal use your plan and retrieved tips** enabled; readable answer on screen.
4. **Guardrails** — prompt that triggers refusal before any LLM call (e.g. diagnosis-style wording).
